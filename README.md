## Usage
## :gear: Setup your own

```bash
# Clone repo
$ git clone git@github.com:ThucNguyen007/django-playground.git

$ cd thuchome

# Create a virtualenv (Optional but reccomended)
$ virtualenv .

# Install all dependencies
$ pip install -r requirements.txt

# Runserver of django backend
$ python manage.py runserver

# Open another Windows terminal after finishing compilling code
$ python manage.py makemigrations
$ python manage.py migrate
